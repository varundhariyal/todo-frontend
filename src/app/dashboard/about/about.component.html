About TODO

Overview of TODO Web App

User-Management

Signup- New account creation. Login- User login. Forgot Password- Password reset functionality.

FriendZone

User can send friend request to other users on platform , friend request sent /received /accepted notifications are reflected in real time handle via sockets.

SINGLE-USER-TODO

There is a dedicated section for performing crud operations over todos.User can create/delete a todo list add/edit/delete task and add/delete subtask to the list.

MULTI-USER-TODO

Only friends can create/edit/delete todo task . Friends can edit other todo task under this section and any friend can undo any number of change done in oast to that todo task via keyboard shortcut key(ctrl+z/cmd+z) or via a dedicated undo button.
